User Manager Architecture

Visão geral da arquitetura e segurança da aplicação.

Diagrama da Arquitetura
Tecnologias
  • FastAPI
  • Token JWT
  • Autenticação com OAuth2
  • PostgreSQL
  • Nginx
  • Docker
  • AWS EC2
  • HTTPS (Let's Encrypt)
Fluxo da Aplicação

Cliente → HTTPS → Nginx → API interna → Banco de Dados

Arquitetura

O Nginx atua como proxy reverso, servindo o frontend e encaminhando requisições para a API interna. A API roda em container Docker e se comunica exclusivamente com o banco PostgreSQL.

Segurança
  • HTTPS com TLS
  • API não exposta
  • Banco isolado
  • Autenticação JWT
  • CORS restrito
