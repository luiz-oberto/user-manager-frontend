Fluxo

Cliente → HTTPS → Nginx → API → Banco de Dados

Arquitetura

O Nginx atua como proxy reverso, servindo o frontend e encaminhando requisições para a API interna em container Docker, que se comunica com o PostgreSQL.

Segurança
  • HTTPS com TLS
  • API isolada
  • Banco protegido
  • JWT
  • CORS restrito
